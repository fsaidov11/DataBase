Для выполнения данного задания, мы начнем с описания предметной области на основе предоставленного текста:

### Советники Диаспара замерли в ожидании, и их собственная тревога с минуты на минуту росла по мере того, как продолжался этот безмолвный разговор. Но вот глава делегации очнулся от транса и с извиняющимся видом повернулся к председателю.

Предметная область: Встреча делегации советников в городе Диаспар.

### Сущности:

***Делегация*** - это группа людей, представляющих определенную организацию или группу интересов. 

У делегации есть следующие атрибуты:

Идентификатор (***delegation_id***) - уникальный идентификатор делегации.
Глава делегации (***delegation_leader***) - человек, который является главой делегации.
Состояние (***delegation_status***) - текущее состояние делегации (например, "ожидание").

***Советник*** - это член делегации, который участвует во встрече в городе Диаспар. У советника есть следующие атрибуты:
Идентификатор (***advisor_id***) - уникальный идентификатор советника.
Имя (***advisor_name***) - имя советника.
Тревога (***advisor_anxiety***) - уровень тревоги советника.

***Город Диаспар*** - место, где происходит встреча с делегацией советников.

### Связи:

Делегация состоит из советников (один ко многим).
Встреча происходит в городе Диаспар.

## Инфологическая модель:

**Delegation**

- id (PK)
- delegation_leader
- emotional_status

**Advisor**

- id (PK)
- delegation_id (FK)
- advisor_name
- advisor_anxiety

**City**

- name

**Meeting**

- id (PK)
- city_id (FK) REFERENCE (**City**)

- second_delegation_id (FK)

**Delegation**-**Advisor**(один ко многим): delegation_id (в **Advisor**) - foreign key.
